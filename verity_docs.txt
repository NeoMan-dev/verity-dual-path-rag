Nexus RAG System V2.0 Documentation

Overview:
The Nexus system represents a premium upgrade to enterprise workflow AI. It natively supports parsing and embedding of PDF, CSV, and TXT files immediately upon upload.

Capabilities:
- Intelligent chunking and local embeddings using All-MiniLM.
- A beautiful glassmorphism dark mode user interface.
- Action triggers that allow users to immediately act upon insights (like opening documentation, drafting emails, or copying templates).
- Highly confident, strictly grounded answers with no hallucinations. Everything must be sourced from context.
